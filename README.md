# Pokémon Site 2.0

Projeto de site estático desenvolvido com **HTML, CSS e JavaScript**, como evolução direta do Pokémon Site 1.0.  
Esta versão foca em **melhorias visuais**, **acessibilidade**, **experiência do usuário** e **animações**.

O projeto é fictício e foi criado exclusivamente para fins educacionais e de portfólio.

---

## Visão Geral

O Pokémon Site 2.0 apresenta um layout mais refinado e recursos adicionais em relação à versão anterior, incluindo:

- Interface visual aprimorada
- Animações ao rolar a página
- Recursos avançados de acessibilidade
- Melhor organização do código
- Experiência mais fluida para o usuário

---

## Novidades em relação à versão 1.0

- Implementação de **ScrollReveal** para animações suaves
- Modo de **alto contraste**
- Uso mais consistente de atributos de acessibilidade (`aria-*`)
- Rodapé com ícones de redes sociais
- Melhor organização de estilos e scripts

---

## Acessibilidade

O projeto conta com recursos voltados à inclusão digital, como:

- Botões para aumentar e diminuir o tamanho da fonte
- Alternância de modo **alto contraste**
- Uso de `aria-label`, `aria-expanded` e `tabindex`
- Estrutura semântica e navegação clara por seções

---

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5  
- Bootstrap Icons  
- ScrollReveal.js
